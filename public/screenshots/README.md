## Tips for high-quality captures

- Press **F12 → Toggle device toolbar → Responsive → 1600×900** for consistent dimensions
- Or on Mac: **Cmd+Shift+5** → window selection
- Or on Windows: **Snipping Tool → Window**
- Crop tightly around the viewport — no chrome, no taskbar
- Save as PNG, optionally optimize via [squoosh.app](https://squoosh.app)
